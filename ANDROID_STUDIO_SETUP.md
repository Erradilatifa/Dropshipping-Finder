# Configuration Android Studio

## Ouvrir le projet dans Android Studio

1. Ouvrez Android Studio
2. Cliquez sur "Open an existing project"
3. Naviguez vers: `c:\Users\DELL\dropshipping-finder\frontend`
4. Sélectionnez le dossier `frontend` et cliquez "OK"

## Configuration requise

### SDK Flutter
- Assurez-vous que Flutter SDK est configuré dans Android Studio
- File > Settings > Languages & Frameworks > Flutter
- Définir le chemin Flutter SDK

### Émulateur Android
- Tools > AVD Manager
- Créer un nouvel émulateur si nécessaire
- Recommandé: Pixel 6 API 33

## Démarrer sur Android

1. Ouvrez un terminal dans Android Studio
2. Exécutez: `flutter run`
3. Ou utilisez le bouton Run dans l'IDE
